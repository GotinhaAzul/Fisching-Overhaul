# Fisching-Overhaul

Simple Fisching text game overhaul.

## Como executar

```bash
python main.py
```

## Estrutura

- `main.py`: ponto de entrada.
- `fisching/game.py`: loop principal do jogo.
- `fisching/fishing.py`: lógica de pesca e sequências de teclas.
- `fisching/inventory.py`: controle do inventário.
- `fisching/models.py`: modelos de dados.
- `fisching/utils.py`: utilidades de formatação.
